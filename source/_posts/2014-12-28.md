今天搞了一上午的博客，就弄了一点点东西。  
这里是[教程](http://zipperary.com/),当然更[详细](http://ijiaober.github.io/categories/hexo/)的在这。
hexo版本设计之后各模块都拆分出来，除了官网首页里列的一些命令：
```
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
```
还需`npm install hexo-deployer-git --save`安装提交到git的模块
